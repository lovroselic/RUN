About:

R.U.N. (Run Upward Nerd), is heavily influenced by the legenary H.E.R.O. (standing for Helicopter Emergency Rescue Operation) released in 1984. This remains one of my favorite games of the early 80's and I wanted to make a version of it for a long time. The main difference to H.E.R.O. is that in the R.U.N. you need to save yourself and run upwards. And you use jetpac instead of helicopter-pack (due to my drawing skill limitations).

Another influence is O'Riley's Mine, released in 1983 from which the idea of imminent flood is taken, to motivate you to move faster ...

R.U.N. is written in JavaScript, using JQuery and numerous private libraries written by myself.
